## ğŸ’­ Why I Built This

**As someone managing CRM platforms serving 3,000+ users across 22 countries, I've seen how expensive and bloated enterprise data tools can be.**

- **Tableau Prep:** $25K-$75K annually
- **Alteryx:** $15K-$50K per seat
- **Both require:** Cloud uploads, lengthy implementations, extensive training

**I built dataphreak to prove that professional-grade data quality tools don't need enterprise licensing or cloud dependencies. The best leaders never stop coding.**

This project specifically explores:

- âœ… **Fundamental algorithms** - Levenshtein distance, statistical profiling, join operations
- âœ… **Single-file deployment** - Works forever (download today, works in 10 years)
- âœ… **Privacy by design** - Processing locally isn't just a feature, it's a philosophy
- âœ… **Zero dependencies** - Embedded SheetJS means true offline capability
- âœ… **Performance at scale** - 1M+ rows without external engines

**The privacy angle matters.** Data professionals shouldn't have to upload sensitive customer information to third-party servers just to find duplicates or assess quality. Processing locally isn't just a featureâ€”it's a philosophy about data ownership and control.
